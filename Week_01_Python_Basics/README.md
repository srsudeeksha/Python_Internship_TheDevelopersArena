## 📂 Files in this Folder

### 1️⃣ \`calculator.py\`
A simple calculator that performs:
- Addition
- Subtraction
- Multiplication
- Division (with zero-division handling)

**How to run:**

python calculator.py

You will be prompted to enter:
1. Operation (+, -, *, /)
2. Two numbers

### 2️⃣ \`even_odd_checker.py\`

**How to run:**

python even_odd_checker.py
